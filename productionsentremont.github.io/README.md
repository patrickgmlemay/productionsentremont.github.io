# Productions Entremont

Website for Productions Entremont — short film production company, Montréal.

## Setup

1. Add your video file to `videos/`:
   - `Entremont_Introduction_Animation.mov`

2. Add your EPK PDF to `press/`:
   - `FIX_EPK_Entremont_2024.pdf`

3. Delete the `PUT_*_HERE.txt` placeholder files.

4. Push to GitHub — the site deploys automatically via GitHub Pages.

## Live URL

`https://productionsentremont.github.io`

## Notes

- GitHub has a 100MB file size limit per file.
- If your video exceeds 100MB, host it on Vimeo and update the src in index.html.
